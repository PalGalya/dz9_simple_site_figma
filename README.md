# Simple Site SCSS

## Структура проекта
```
src/
└── scss/
    ├── abstracts/       # Общие инструменты и хелперы
    │   ├── _index.scss
    │   ├── _variables.scss
    │   ├── _mixins.scss
    │   ├── _mixins-media.scss
    │   └── _extends.scss
    │
    ├── base/            # Базовые стили
    │   ├── _index.scss
    │   ├── _normalize.scss
    │   ├── _typography.scss
    │   └── _common.scss
    │
    ├── components/      # Компоненты
    │   ├── _index.scss
    │   ├── _navbar.scss
    │   ├── _content.scss
    │   └── _experience-item.scss
    │
    ├── layout/         # Макет и секции
    │   ├── _index.scss
    │   ├── _header.scss
    │   ├── _footer.scss
    │   ├── _section.scss
    │   ├── _experience.scss
    │   ├── _video.scss
    │   ├── _about.scss
    │   └── _works.scss
    │
    └── style.scss      # Главный файл
```

## Використані технології
- SCSS
- BEM методологія
- Адаптивний дизайн
- Mobile First підхід

## Особливості
- Використання міксинів
- Адаптивна верстка
- Модульна структура (7-1 патерн)
